# Changelog

## 1.0.0 (2026-04-08)

### Funzionalità
- Riconoscimento facciale in tempo reale tramite modello SVM personalizzato
- Interfaccia web integrata in Home Assistant (Ingress)
- Supporto telecamere HA e URL snapshot diretti
- Entità MQTT auto-discovery (switch, sensor, binary_sensor, camera)
- Upload modello classificatore via interfaccia web
- Salvataggio volti ritagliati per ri-addestramento
- Predisposizione per riconoscimento targhe (futuro)
